# SkillScope AI

## 📌 Project Overview

SkillScope AI is a data analytics project focused on analyzing Data Analyst job postings to identify:

- Most demanded skills
- Salary trends
- Hiring locations
- Experience-level demand
- Top hiring companies

This project demonstrates an end-to-end data analytics workflow using Excel, SQL, Python, and Power BI.
